## How It Works on Windows

### Path Handling

The code uses **cross-platform path methods**:

```python
# All paths in the code use:
from pathlib import Path
import os

# Database (relative path)
sqlite3.connect("proctoring.db")  # Works on any OS

# Models (dynamic resolution)
backend_dir = Path(__file__).resolve().parents[1]
model_path = str(backend_dir / 'models' / 'yolov8n.pt')
# Auto-converts "/" to "\" on Windows

# Evidence storage (OS-aware)
os.path.join('data', 'evidence')  # Uses "\" on Windows, "/" on Unix
```

### Virtual Environment

- **macOS/Linux**: `./backend_env/bin/python`
- **Windows**: `.\backend_env\Scripts\python.exe`

The batch script handles both automatically.

### Log Files

- **macOS/Linux**: `/tmp/proctoring_server.log`
- **Windows**: `C:\Users\[User]\AppData\Local\Temp\proctoring_server.log`

Uses `%TEMP%` environment variable on Windows.
